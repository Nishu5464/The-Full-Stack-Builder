# 🚀 The Full Stack Builder

A full-stack **Task Management Application** built using **Flutter (Frontend)** and **FastAPI (Backend)**.
This project demonstrates end-to-end development including UI, API integration, and database handling.

---

## 📌 Features

* ✅ Create, update, and delete tasks
* ✅ Task status tracking (To-Do, In Progress, Done)
* ✅ Task dependency/blocking system
* ✅ REST API integration
* ✅ Clean and responsive Flutter UI
* ✅ Backend powered by FastAPI with SQLite

---

## 🛠️ Tech Stack

### 🔹 Frontend

* Flutter
* Dart

### 🔹 Backend

* FastAPI
* Python
* SQLAlchemy
* SQLite

## ⚙️ Setup Instructions

### 🔹 1. Clone the Repository

```bash
git clone https://github.com/Nishu5464/The-Full-Stack-Builder_N.git
cd The-Full-Stack-Builder_N
```

---

### 🔹 2. Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

👉 Backend runs on:
http://127.0.0.1:8000

👉 API Docs:
http://127.0.0.1:8000/docs

---

### 🔹 3. Frontend Setup

```bash
cd frontend
flutter pub get
flutter run -d chrome
```
